### 작성 

* 마크다운을 기본으로 작성합니다. 
* 세미나 내용 준비 및 개인 공부, 프로젝트, 그날의 느낌을 바로 작성하여 올릴 수 있도록 합니다. 
* 프로젝트 폴더는 영문으로 작성하나 파일 이름은 한글이 포함될 수 있습니다. 
* contributions을 채우는 것에 집착하지 않고 자유롭게 나의 내용을 기록할 수 있도록 합니다.
